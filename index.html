Folyamatkezelés

1. Új ügyfél rögzítése

Új partner adatainak felvétele, kötelező mezők ellenőrzése, majd mentés az ügyféltörzsbe.

2. Ügyféladat módosítása

3. Riport frissítés

Adatok frissítése, ellenőrzés, majd dashboard nézet újratöltése.

4. Backend API kapcsolat

A frontend később Access vagy MS SQL backendhez kapcsolható API-n keresztül.
